# WB_WildHack

`notebooks/baseline_template.ipynb` contains the original baseline.

`notebooks/solo_colab_experiments.py` is the stronger solo-track workflow for Colab A100:
- large EDA with saved artifacts,
- feature-engineered CatBoost and XGBoost direct multi-horizon models,
- optional zero-shot TimesFM and Timer-family forecasts,
- validation-calibrated blending and submission export.
